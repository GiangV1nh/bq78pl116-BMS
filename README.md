- Loại tế bào: Li-Ion
- Điện áp tế bào: 2.75V - 4.5V
- Điện áp pin: 8.25V - 12.6V

- Dòng xả tối đa: 11.2A

- Dòng cân bằng: 143mA
- Giao tiếp: SMBus
